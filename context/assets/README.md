# Brand Assets

Place your brand assets in this folder. These are referenced by skills during ad generation.

## Expected Files
- `logo.png` — Primary brand logo (transparent background, min 500px wide)
- `logo-white.png` — White/light version for dark backgrounds (optional)
- `product-*.png` — Product images (e.g., `product-hero.png`, `product-serum.png`)
- `lifestyle-*.jpg` — Lifestyle photography for ad backgrounds

## Supported Formats
JPG, PNG, WebP

## Usage
Skills reference these files automatically via paths in `context/brand-voice.md`.
Update the `Logo path` field in the brand kit to match your actual filename.
