# Reference Ads Library

Place your best-performing ads here. These are automatically loaded by creative skills as examples of on-brand advertising.

## How It Works
When any creative skill runs, it checks this folder and describes the reference ads to the AI as "this is what on-brand advertising looks like for this brand." This provides a soft style influence without manual input.

## Naming Convention
- `meta-[description].png` — Facebook/Instagram ads
- `tiktok-[description].png` — TikTok ads
- `static-[description].png` — Static image ads
- `ugc-[description].png` — UGC-style content

## Tips
- Include your top 3-5 performers
- Mix formats: at least one static, one video thumbnail, one UGC-style
- Update quarterly as creative evolves
- Remove underperformers — only keep what represents your best work
