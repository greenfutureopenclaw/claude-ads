## Ad Templates

### TEMPLATE A — Gamified Engagement (Match the Shadow)
**Best for:** Driving comments and saves via interactive format

- **Aspect ratio:** 1:1 Square
- **Canvas split:** Top 60% focus area, Bottom 40% selection grid
- **Background:** Solid light pastel lavender/grey (#F3F5F9)

**Elements:**
- **Headline:** "MATCH THE SHADOW!" — Bold Sans-Serif, All-Caps, Top center
  - Colors: #2D3E50 (Navy) + #E1306C (Pink)
- **Main focus:** Product silhouette as solid white cutout inside a horizontal pink block (#D7385E)
- **Selection grid:** 3 equal square product photos with circular red badges (A, B, C) at top-center
  - Content: Real-life lifestyle photography (mugs, plushies, bouquets)

### TEMPLATE B — AI UI Mockup (ChatGPT Meta-Humor)
**Best for:** Validating product value through social proof + relatability

- **Aspect ratio:** 4:5 or 1:1 (Mobile Screen UI)
- **Background:** Clean white app interface, light grey dividers

**Elements:**
- **Header:** "ChatGPT 5.2" — hamburger menu left, centered model name, system sans-serif
- **User bubble:** Soft pink (#F0B4D1) — studio gift box image + text "make this the perfect gift for her"
- **AI response bubble:** Left-aligned — "it already is." — with thumbs up/down, copy, speaker icons
- **Input field:** Rounded pill-shape bottom bar with "Ask anything." placeholder + plus icon

### TEMPLATE C — Promo Lifestyle (Topical Discount Banner)
**Best for:** High-conversion promotional posts tied to current events

- **Aspect ratio:** 1:1 Square
- **Background:** Soft gradient or solid off-white (#F8F9FB)

**Elements:**
- **Main headline:** "NAGMAMAHAL NA RAW GAS. 'WAG KA NANG LUMABAS."
- **Sub-headline:** "KAILANGAN MO NG REGALO? TRY US."
  - Font: Bold Sans-Serif, generous kerning
  - Colors: #2D3E50 (primary) + #E1306C (accent)
- **Discount badge:** Rounded white square with drop shadow
  - Main text: "₱250 OFF"
  - Promo code box: #D7385E background — "USE CODE: 250XPO" in white bold
  - Small grey terms text (min spend + validity)
- **Lifestyle image:** Smiling woman, half-body, hand-held product (e.g., crochet bouquet), right side, slightly overlapping badge
  - Lighting: Natural, bright, soft shadows
